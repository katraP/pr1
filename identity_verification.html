Confirm the authorization code

  1. Call your bank/credit card company and inquire about the Authorization Code for the transaction under the name WRITING SERVICES.
    Authorization Code is a 6-7 digit code, which consists of numbers (examples - 281435; 0015154) or combination of numbers and a letter (examples - 0r61145; 005525c);
  2. Once you have the required information, confirm the Authorization Code by contacting Customer Support (e-mail, message, call, chat).
